# 🏡 Homestay Smart Booking System

## 📌 Giới thiệu dự án
Hệ thống **đặt phòng và quản lý Homestay thông minh** tích hợp AI dự đoán giá theo thời gian.

---

## 🎯 Mục tiêu
- Cho phép người dùng tìm kiếm & đặt phòng online  
- Quản lý phòng, booking, khách hàng  
- Tích hợp AI dự đoán giá phòng  
- Hỗ trợ chủ homestay tối ưu doanh thu  

---

## 🧠 Công nghệ sử dụng
- **Frontend:** ReactJS  
- **Backend:** ASP.NET Core Web API  
- **Database:** MySQL / SQL Server   
- **Tool:** GitHub, Postman  

---
